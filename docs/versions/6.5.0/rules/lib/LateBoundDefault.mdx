<h1 class="page-title" id="modules.LateBoundDefault">LateBoundDefault</h1>

<!-- {% raw %} -->

Represents a late-bound default attribute value of type 'Label'. The value of a LateBoundDefault is only resolvable in the context of a rule implementation function, and depends on the current build configuration. For example, a LateBoundDefault might represent the Label of the java toolchain in the current build configuration. <p>See <a href="globals.html#configuration_field">configuration_field</a> for example usage.
